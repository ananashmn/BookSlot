BookSlot Setup

1Create a Firebase project

This takes about 2 minutes and is completely free.
1. Go to console.firebase.google.com
2. Click Add project → name it (e.g. "bookslot") → click through
3. Once created, click Firestore Database in the left sidebar
4. Click Create database → choose Start in test mode → select any region → Done
5. Go to Project settings (gear icon) → scroll to Your apps → click the </> (Web) icon → register app → copy the config object

2Paste your Firebase config

Find it in Project Settings → Your apps → SDK setup → Config.
